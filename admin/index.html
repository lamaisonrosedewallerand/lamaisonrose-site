Chargement de l'administration

L'interface Decap CMS est en train de se lancer. Si la connexion GitHub n'est pas encore configuree dans Vercel, la page restera accessible mais la connexion editeur ne pourra pas encore aboutir.

Cette administration n'utilise pas de mot de passe local stocke sur le site : l'acces passe par GitHub OAuth et peut etre limite a des comptes GitHub autorises.
